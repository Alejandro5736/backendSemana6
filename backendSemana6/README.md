# Minimarket Plus - Backend

Proyecto desarrollado para la asignatura de Desarrollo Backend II. Este repositorio contiene la implementación del backend con pruebas unitarias.

## Tecnologías utilizadas
- Java 21
- Maven
- JUnit 5
- JaCoCo (Cobertura de pruebas)

## Descripción
Este proyecto incluye:
1. Pruebas unitarias para los servicios del negocio.
2. Configuración de plugins para análisis de calidad (JaCoCo).
3. Estructura basada en buenas prácticas de arquitectura backend.

## Cómo ejecutar
1. Clona este repositorio.
2. Ejecuta `mvn clean test` en la terminal para correr las pruebas y generar el reporte de cobertura.
